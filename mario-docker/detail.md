# 🚀 Projet 01 : Créer des conteneurs et jouer à Mario

**Objectifs :** Déploiement initial de conteneurs et découverte des commandes de base.

**Réalisations :**
* Lancement du conteneur de test `hello-world` pour vérifier la bonne communication avec le daemon Docker.
* Recherche, téléchargement (`pull`) et déploiement du jeu Mario (`sevenajay/mario:latest`) sur le port local `4545`.
* Déploiement d'un serveur web Nginx sur le port `8080` en mode détaché (`-d`).
* Navigation et interaction dans un shell interactif (`bash`) à l'intérieur du conteneur Nginx.
* Nettoyage de l'environnement avec `docker system prune`.
